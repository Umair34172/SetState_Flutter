# setState Example App

A simple Flutter app demonstrating how setState works.

## Features
- Counter increment using setState
- Theme toggle (Light/Dark)
- Real-time UI update

## What is setState?
setState is used to update the UI when data changes inside a StatefulWidget.

## Technologies
- Flutter
- Dart 

## How to Run
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Output
- Counter increases on button press
- Theme changes dynamically
